# Welcome

This is a sample blog post to demonstrate the Jekyll structure.

## What is Jekyll?

Jekyll is a static site generator that takes Markdown files and converts them into a complete website.

## Directory Structure

This site follows the official Jekyll directory structure as documented at [jekyllrb.com/docs/structure/](https://jekyllrb.com/docs/structure/).

- `_config.yml` - Site configuration
- `_data/` - Data files for site variables
- `_drafts/` - Unpublished posts
- `_includes/` - Reusable HTML snippets
- `_layouts/` - Page templates
- `_posts/` - Blog posts
- `_sass/` - Sass partials
- `_site/` - Generated site (not committed)
- `assets/` - Static assets like CSS, JS, images
- `index.md` - Homepage
- Other `.md` files - Additional pages
